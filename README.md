## Описание проекта

**Fleet Management System** — это консольное приложение на языке C++ с 
использованием базы данных SQLite, предназначенное для автоматизации 
работы автопарка.

Приложение позволяет:
- вести учёт автомобилей, водителей и выполненных заказов;
- контролировать загрузку машин (проверка грузоподъёмности);
- автоматически рассчитывать заработную плату водителей (20% от стоимости 
рейса);
- разграничивать права доступа: администратор управляет всеми данными, 
водитель видит только свои заказы и информацию о закреплённом автомобиле.

Проект реализован с соблюдением принципов CI/CD: автоматическая сборка, 
тестирование и проверка покрытия кода настроены через GitHub Actions.

## Функциональные возможности

### Для администратора
- ✅ Просмотр, добавление, редактирование и удаление записей об 
автомобилях, водителях и заказах.
- ✅ Формирование отчётов:
  - список заказов по водителю за указанный период;
  - общий пробег и масса грузов по каждой машине;
  - статистика по каждому водителю (количество поездок, масса грузов, 
сумма заработка);
  - водитель с наименьшим количеством поездок;
  - автомобиль с наибольшим общим пробегом.
- ✅ Расчёт зарплаты всех водителей за произвольный период с сохранением 
результатов в отдельную таблицу.

### Для водителя
- 🔒 Просмотр только своих заказов.
- 🔒 Информация о закреплённом автомобиле.
- 🔒 Расчёт собственного заработка за указанный период.
